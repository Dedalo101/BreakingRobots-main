
# BreakingRobots

Breaking Robots Free improvisation on the themes of assimilation and disintegration to promote a human awakening process through music and performance that seeks awareness and solutions to dehumanising social conditions and behaviour.

## About

An experimental electronic collective based in Brighton, UK, dedicated to pushing the boundaries of traditional performance art. Our music is a high-energy fusion of industrial rhythms, psychedelic synth-work, and avant-garde electronic textures, often creating an immersive, ritualistic atmosphere during live sets.

## Tech Stack

- **Framework**: React + Vite
- **Graphics**: WebGL shaders for generative morphing visuals
- **Deployment**: GitHub Pages (static site)

## Development

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Build for production: `npm run build`

## Deployment

Push to the `main` branch to deploy via GitHub Pages.
